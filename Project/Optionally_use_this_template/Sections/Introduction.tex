\section{Introduction}\label{sec:intro}
Information Retrieval (IR) aims to retrieve documents that satisfy a user's information need. 
In this project, we build an IR system for the Cranfield dataset using the TF-IDF Vector Space Model. 
Documents and queries are represented as weighted term vectors, and cosine similarity is used for ranking.
The project first implements a baseline VSM system and then improves it using linguistic preprocessing, 
smoothed TF-IDF weighting, pseudo-relevance feedback, and an inverted index for efficient retrieval.

The main problem with a purely lexical VSM system is that it rewards exact term overlap rather than meaning.
This is useful for technical collections such as Cranfield, but it also creates failures when a relevant document
uses a different word form, when a query contains out-of-vocabulary terms, or when a shared term has more than one
sense. The proposed system therefore keeps the interpretability of TF-IDF while adding light-weight changes that
target the observed weaknesses of the baseline.
